# :tv: LiveCode.com
<a name="readme-top"></a>
LiveCode is a code share App.

✅ Student can shaer live code with is mentor.  
✅ The mentor will see the code block he choose with a read only mode.  
✅ The student will see the code block with the ability to change the code.  
✅ Code changes displayed in real-time (Socket).  
<br/>
:link: Link to the live project: [Link](https://livecode-com.onrender.com/)
